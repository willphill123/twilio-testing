SMS Development Testing - Opt-in Documentation

Last Updated: November 8, 2024

Development Environment Opt-in Process

This documentation outlines the opt-in process for our development and testing environment for SMS messaging services.

Scope of Testing

SMS testing is strictly limited to:

  • Personal phone numbers of the developer
  • Voluntary development team members who have explicitly agreed to receive test messages
  • Test phone numbers provided by the messaging service provider

Opt-in Process

Verbal Consent Protocol:

  1. Development team members are personally informed about the testing purpose
  2. Explicit verbal consent is obtained before adding any number to the testing pool
  3. Participants are informed that:
    • Messages are for development purposes only
    • They can opt out at any time by replying "STOP"
    • Standard message and data rates may apply
    • Testing is temporary and limited in scope

Testing Parameters

All test messages will:

  • Be clearly marked as test messages
  • Include opt-out instructions
  • Be sent during reasonable hours
